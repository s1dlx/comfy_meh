# comfy_meh

Bringing all the merging methods from [meh](https://github.com/s1dlx/meh) to ComfyUI

## Install

```
cd <path to>/ComfyUI/custom_nodes/
git clone https://github.com/s1dlx/comfy_meh
cd comfy_meh
pip install -r requirements.txt
```
